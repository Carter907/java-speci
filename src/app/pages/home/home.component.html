An In-Depth Java Specification Learning Platform

Overview

This application aims to familiarize you with the Java Lang Specification text found on

The Oracle Website

Although these specifications are intimidatingly concise, you can explore this applications to become more familiar with the key concepts. clicking on the question mark will lead you to the questions categories section.

Categories

Categories are units of knowledge representing the Chapters in the specification. For instance, if you want to learn more about how the JVM works, you would click on "Chapter 12. Execution"
